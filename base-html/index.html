HTML-документ— это обычный текстовый документ, может быть создан как в обычном текстовом редактор (Блокнот), так и в специализированном, с подсветкой кода (Notepad++, Visual Studio Code и т.п.). HTML-документ имеет расширение .html.

HTML-документ состоит из дерева HTML-элементов и текста. Каждый элемент обозначается в исходном документе начальным (открывающим) и конечным (закрывающим) тегом (за редким исключением).

Начальный тег показывает, где начинается элемент, конечный — где заканчивается. Закрывающий тег образуется путем добавления слэша / перед именем тега: <имя тега>.... Между начальным и закрывающим тегами находится содержимое элемента — контент.

Например:

1.Контент – Создаёт параграф

2. Контент – Делает полужирное начертание

3. Контент – Курсивное начертание
